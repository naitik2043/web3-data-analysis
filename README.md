# 📊 Web3 Data Analysis

## Overview

This project analyzes the relationship between Bitcoin market sentiment and historical trader performance using Python, Pandas, NumPy, Matplotlib, and Seaborn.

The analysis explores how market sentiment (Fear/Greed) influences trader behavior, profitability, trading volume, and overall performance through Exploratory Data Analysis (EDA) and data visualization.

---

## Project Objectives

- Clean and preprocess historical trading data.
- Create a market sentiment dataset.
- Merge trading and sentiment datasets.
- Perform Exploratory Data Analysis (EDA).
- Visualize trading patterns.
- Analyze trader performance under different market conditions.
- Generate actionable insights for smarter trading strategies.
